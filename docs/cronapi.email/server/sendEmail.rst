Envia e-mail
============

Função que envia e-mail

Parâmetros
~~~~~~~~~~

O método Envia e-mail aceita os seguintes parãmetros para a sua execução:

De

Para

     Pode receber um único ou uma lista de destinatários

Com cópia

     Um único campo texto ou uma lista de destinatários

Com cópia oculta

     Um único campo texto ou uma lista de textos

Assunto

Mensagem

Conteúdo HTML

    Envia mensagem com o conteúdo do HTML

Anexo(s)

     Caminho absoluto do anexo, ou lista com caminhos dos anexos

Hospedeiro SMTP

Porta SMTP

Login

     Login para autenticação no hospedeiro SMTP

Senha

     Senha para autenticação no hospedeiro SMTP

SSL?

    Usa SSL se verdadeiro, senão usa TSL.
