Renomear arquivo
================

Função para renomear arquivo. Exemplo: /MinhaPasta/ArquivoAntigo.txt é renomeado para /MinhaPasta/ArquivoNovo.txt

Parâmetros
~~~~~~~~~~

O método Renomear arquivo aceita os seguintes parãmetros para a sua execução:

Caminho do arquivo

     Exemplo: PASTA_TEMPORARIA/MeuArquivo.txt

Novo nome

     Novo nome do arquivo
