Obter perfil no formato CSV
===========================

Gera um perfil de personalidade no formato CSV para o autor do texto de entrada.

Parâmetros
~~~~~~~~~~

O método Obter perfil no formato CSV aceita os seguintes parãmetros para a sua execução:

A data da versão da API REST.

O nome de usuário.

A senha.

O novo endPoint da API.

Os cabeçalhos padrão para serem usados em todas as solicitações HTTP.

Um objeto Java que especifica os parâmetros da solicitação. Saiba mais

     https://www.ibm.com/watson/developercloud/personality-insights/api/v3/#profile

Indica se os rótulos das colunas devem ser retornados com a resposta CSV.
