Definir valor do atributo
=========================

Função que define o valor do atributo de um elemento

Parâmetros
~~~~~~~~~~

O método Definir valor do atributo aceita os seguintes parãmetros para a sua execução:

Obter identificador

     Identificador (id) a ser obtido do Formulário selecionado

Nome do atributo

     Nome do atributo contido no Identificador. Exemplo:  src='teste' retorna src

Valor do atributo

     Valor do atributo contido no Identificador. Exemplo:  src='teste' retorna teste
