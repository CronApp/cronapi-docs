Obter informações sobre um classificador
========================================

Retorna o status e outras informações sobre um classificador.

Parãmetros
~~~~~~~~~~

O método Obter informações sobre um classificador aceita os seguintes parãmetros para a sua execução:

username

    O nome de usuário.

password

    A senha.

endPoint

    O novo endPoint da API.

headers

    Os cabeçalhos padrão para serem usados em todas as solicitações HTTP.

classifierId

    ID do classificador para consulta.
