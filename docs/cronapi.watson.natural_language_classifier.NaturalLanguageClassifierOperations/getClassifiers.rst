Listar classificadores
======================

Retorna uma matriz vazia se nenhum classificador estiver disponível.

Parãmetros
~~~~~~~~~~

O método Listar classificadores aceita os seguintes parãmetros para a sua execução:

username

    O nome de usuário.

password

    A senha.

endPoint

    O novo endPoint da API.

headers

    Os cabeçalhos padrão para serem usados em todas as solicitações HTTP.
