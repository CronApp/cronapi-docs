Agendar execução
================

Função que agenda comandos a serem executados após um determinado atraso ou de forma periódica.

Parâmetros
~~~~~~~~~~

O método Agendar execução aceita os seguintes parãmetros para a sua execução:

Comandos

     Comandos a serem executados

Tempo inicial

     Tempo inicial necessário até que a função execute pela primeira o(s) comando(s) definido(s)

Tempo para Atualização

     Tempo de espera entre as execuções do(s) comandos(s)

Unidade de Tempo

     Unidade de medida do tempo passado como parâmetro
