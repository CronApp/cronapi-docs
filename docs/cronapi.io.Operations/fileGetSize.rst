Tamanho do arquivo
==================

Função para obter o tamanho total do arquivo. Exemplo, arquivo com 4096 bytes terá como retorno 4096

Parâmetros
~~~~~~~~~~

O método Tamanho do arquivo aceita os seguintes parãmetros para a sua execução:

Arquivo aberto para leitura

     Retorno da função "Abrir arquivo para leitura"
