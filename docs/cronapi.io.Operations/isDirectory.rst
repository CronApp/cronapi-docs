É pasta?
========

Função verificar se é pasta. Retorna verdadeiro se o caminho passado como parâmetro representa um diretório, se não, retorna falso.

Parâmetros
~~~~~~~~~~

O método É pasta? aceita os seguintes parãmetros para a sua execução:

Caminho

     Caminho completo, exemplo PASTA_TEMPORARIA/minhaPasta
