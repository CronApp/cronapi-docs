Obter lista de arquivos
=======================

Função que lista arquivos

Parâmetros
~~~~~~~~~~

O método Obter lista de arquivos aceita os seguintes parãmetros para a sua execução:

Caminho do arquivo

     Caminho completo do arquivo. Exemplo: /MinhaPasta/MeuArquivo.txt ou /MinhaPasta/

Tipo

     Tipo para listar arquivos. Todos: Lista arquivos e diretórios. Diretório: Lista apenas os diretórios, sem arquivos. Arquivo: Lista apenas os arquivos, sem diretórios.
