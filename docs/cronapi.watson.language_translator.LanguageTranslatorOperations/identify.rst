Identificar idioma
==================

Identifique o idioma de uma amostra de texto.

Parãmetros
~~~~~~~~~~

O método Identificar idioma aceita os seguintes parãmetros para a sua execução:

username

    O nome de usuário.

password

    A senha.

endPoint

    O novo endPoint da API.

headers

    Os cabeçalhos padrão para serem usados em todas as solicitações HTTP.

identifyOptions

    Opções Identificação de idioma.  Saiba mais: https://www.ibm.com/watson/developercloud/language-translator/api/v2/#get-model
