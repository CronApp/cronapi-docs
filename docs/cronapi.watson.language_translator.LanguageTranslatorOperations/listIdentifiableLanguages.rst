Idiomas identificáveis
======================

Liste os idiomas que o serviço pode identificar.

Parãmetros
~~~~~~~~~~

O método Idiomas identificáveis aceita os seguintes parãmetros para a sua execução:

username

    O nome de usuário.

password

    A senha.

endPoint

    O novo endPoint da API.

headers

    Os cabeçalhos padrão para serem usados em todas as solicitações HTTP.
