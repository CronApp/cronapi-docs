Traduzir
========

Traduzir texto para uma linguagem.

Parãmetros
~~~~~~~~~~

O método Traduzir aceita os seguintes parãmetros para a sua execução:

username

    O nome de usuário.

password

    A senha.

endPoint

    O novo endPoint da API.

headers

    Os cabeçalhos padrão para serem usados em todas as solicitações HTTP.

translateOptions

    Opções de tradução. Mais: https://www.ibm.com/watson/developercloud/language-translator/api/v2/#translate
