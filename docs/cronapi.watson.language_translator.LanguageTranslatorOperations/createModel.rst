Criar modelo
============

Carregue um arquivo de glossário TMX em cima de um domínio para personalizar um modelo de tradução.

Parãmetros
~~~~~~~~~~

O método Criar modelo aceita os seguintes parãmetros para a sua execução:

username

    O nome de usuário.

password

    A senha.

endPoint

    O novo endPoint da API.

headers

    Os cabeçalhos padrão para serem usados em todas as solicitações HTTP.

createModelOptions

    As opções. Saiba mais: https://www.ibm.com/watson/developercloud/language-translator/api/v2/#create-model
