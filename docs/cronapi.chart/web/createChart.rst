Criar gráfico
=============

Função que cria um gráfico

Parâmetros
~~~~~~~~~~

O método Criar gráfico aceita os seguintes parãmetros para a sua execução:

Id

     Id do gráfico (canvas)

Tipo de gráfico

     Tipo de gráfico a ser criado

Legenda

     Legenda do gráfico

Opções

     Opções do gráfico

Série(s) do gráfico

     Conjunto de séries do gráfico
