Desenha circulo
===============

Função que desenha uma área circular no mapa

Parâmetros
~~~~~~~~~~

O método Desenha circulo aceita os seguintes parãmetros para a sua execução:

Id do mapa

Id do circulo

Ponto central

Raio em metros

Cor do preenchimento

Cor do contorno

Opacidade do preenchimento

Opções avançadas
