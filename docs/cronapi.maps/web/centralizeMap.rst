Centraliza o mapa
=================

Função que centraliza o mapa nas coordenadas informadas.

O método Centraliza o mapa aceita os seguintes parâmetros para a sua execução:

* `Id do mapa<#mapId>`_
* `Latitude<#mapId>`_
* `Longitude<#mapId>`_
