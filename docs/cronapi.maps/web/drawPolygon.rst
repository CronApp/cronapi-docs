Desenha polígono
================

Função que desenha uma área circular no mapa

Parâmetros
~~~~~~~~~~

O método Desenha polígono aceita os seguintes parãmetros para a sua execução:

Id do mapa

Id do polígono

Endereço da coordenada

Cor do preenchimento

Opacidade do preenchimento

Cor do contorno

Opacidade do contorno
