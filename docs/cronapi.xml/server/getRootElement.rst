Obtém elemento raiz
===================

Função que recebe o documento XML ou um elemento XML e retorna o elemento raiz do documento.

Parâmetros
~~~~~~~~~~

O método Obtém elemento raiz aceita os seguintes parãmetros para a sua execução:

Documento XML

    Documento XML ou  elemento XML para obter o elemento raiz do documento.
