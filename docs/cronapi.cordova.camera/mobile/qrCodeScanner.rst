Ler código de barras
====================

Função que lê alguns formatos conhecidos e retorna o conteúdo. Exemplo: QR_CODE

Parâmetros
~~~~~~~~~~

O método Ler código de barras aceita os seguintes parãmetros para a sua execução:

Formato

     Formato a ser buscado

Mensagem

     Mensagem exibida enquanto a camera esta aberta

Sucesso

Erro
