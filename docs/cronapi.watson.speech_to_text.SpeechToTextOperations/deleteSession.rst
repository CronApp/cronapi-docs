Eliminar uma sessão
===================

Exclui uma sessão existente e seu mecanismo.

Parãmetros
~~~~~~~~~~

O método Eliminar uma sessão aceita os seguintes parãmetros para a sua execução:

username

    Nome do usuário.

password

    A senha.

endPoint

    O novo endPoint da API.

headers

    Nome do cabeçalho que será usado na requisição HTTP.

session

    Um objeto Java que identifica a sessão que deve ser excluída.
