Treinar um modelo de linguagem personalizado
============================================

Inicia o treinamento de um modelo de linguagem personalizado com novos corpora, palavras personalizadas ou ambos.

Parãmetros
~~~~~~~~~~

O método Treinar um modelo de linguagem personalizado aceita os seguintes parãmetros para a sua execução:

username

    Nome do usuário.

password

    A senha.

endPoint

    O novo endPoint da API.

headers

    Nome do cabeçalho que será usado na requisição HTTP.

customizationId

    O GUID do modelo de linguagem personalizado que deve ser treinado.

wordTypeToAdd

    O tipo de palavras do recurso do modelo de palavras da linguagem personalizada para treinar o modelo.
