Excluir um corpus
=================

Exclui um corpus existente de um modelo de idioma personalizado.

Parãmetros
~~~~~~~~~~

O método Excluir um corpus aceita os seguintes parãmetros para a sua execução:

username

    Nome do usuário.

password

    A senha.

endPoint

    O novo endPoint da API.

headers

    Nome do cabeçalho que será usado na requisição HTTP.

customizationId

    O GUID do modelo de idioma personalizado a partir do qual um corpus deve ser excluído.

corpusName

    O nome do corpus que deve ser excluído do modelo de idioma personalizado.
