Redefinir um modelo de idioma personalizado
===========================================

Redefine um modelo de idioma personalizado removendo todos os corpos e as palavras do modelo.

Parãmetros
~~~~~~~~~~

O método Redefinir um modelo de idioma personalizado aceita os seguintes parãmetros para a sua execução:

username

    Nome do usuário.

password

    A senha.

endPoint

    O novo endPoint da API.

headers

    Nome do cabeçalho que será usado na requisição HTTP.

customizationId

    O GUID do modelo de idioma personalizado que deve ser redefinido.
