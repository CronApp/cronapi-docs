Adicionar palavras personalizadas
=================================

Adiciona uma ou mais palavras personalizadas a um modelo de idioma personalizado.

Parãmetros
~~~~~~~~~~

O método Adicionar palavras personalizadas aceita os seguintes parãmetros para a sua execução:

username

    Nome do usuário.

password

    A senha.

endPoint

    O novo endPoint da API.

headers

    Nome do cabeçalho que será usado na requisição HTTP.

customizationId

    O GUID do modelo de idioma personalizado ao qual as palavras devem ser adicionadas.

words

    Uma lista de objetos Java separada por vírgulas, cada uma das quais fornece informações sobre uma palavra personalizada a ser adicionada.
