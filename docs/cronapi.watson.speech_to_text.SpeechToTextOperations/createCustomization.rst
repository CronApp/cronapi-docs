Crie um modelo de idioma personalizado
======================================

Cria um novo modelo de idioma personalizado para um modelo de base especificado.

Parãmetros
~~~~~~~~~~

O método Crie um modelo de idioma personalizado aceita os seguintes parãmetros para a sua execução:

username

    Nome do usuário.

password

    A senha.

endPoint

    O novo endPoint da API.

headers

    Nome do cabeçalho que será usado na requisição HTTP.

name

    O nome do modelo de idioma personalizado.

baseModel

    Um objeto Java que identifica o modelo base que deve ser personalizado pelo novo modelo.

description

    Uma descrição do novo modelo de idioma personalizado.
