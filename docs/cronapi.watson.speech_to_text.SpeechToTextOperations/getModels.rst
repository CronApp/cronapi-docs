Obter modelos
=============

Recupera uma lista de todos os modelos de idiomas disponíveis para uso com o serviço.

Parãmetros
~~~~~~~~~~

O método Obter modelos aceita os seguintes parãmetros para a sua execução:

username

    Nome do usuário.

password

    A senha.

endPoint

    O novo endPoint da API.

headers

    Nome do cabeçalho que será usado na requisição HTTP.
