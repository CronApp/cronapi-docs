Listar de modelos de idiomas personalizados
===========================================

Lista informações sobre todos os modelos de idiomas personalizados que pertencem a uma instância do serviço.

Parãmetros
~~~~~~~~~~

O método Listar de modelos de idiomas personalizados aceita os seguintes parãmetros para a sua execução:

username

    Nome do usuário.

password

    A senha.

endPoint

    O novo endPoint da API.

headers

    Nome do cabeçalho que será usado na requisição HTTP.

language

    O identificador do idioma para o qual os modelos de linguagem personalizados devem ser retornados (por exemplo, pt-BR).
