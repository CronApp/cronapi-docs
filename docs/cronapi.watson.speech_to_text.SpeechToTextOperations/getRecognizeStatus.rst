Obter status
============

Verifica se uma sessão específica pode aceitar outro pedido de reconhecimento.

Parãmetros
~~~~~~~~~~

O método Obter status aceita os seguintes parãmetros para a sua execução:

username

    Nome do usuário.

password

    A senha.

endPoint

    O novo endPoint da API.

headers

    Nome do cabeçalho que será usado na requisição HTTP.

session

    Um objeto Java que identifica a sessão cujo status deve ser verificado.
