Recuperar uma lista de classificadores personalizados
=====================================================

Recupere uma lista de classificadores criados pelo usuário.

Parãmetros
~~~~~~~~~~

O método Recuperar uma lista de classificadores personalizados aceita os seguintes parãmetros para a sua execução:

versionDate

    A data da versão da API REST.

apiKey

    A chave da API.

endPoint

    O novo endPoint da API.

headers

    Nome do cabeçalho que será usado na requisição HTTP.

options

    Opções do Recuperar uma lista de classificadores personalizados. Saiba mais: https://www.ibm.com/watson/developercloud/visual-recognition/api/v3/#retrieve_a_list_of_classifiers
