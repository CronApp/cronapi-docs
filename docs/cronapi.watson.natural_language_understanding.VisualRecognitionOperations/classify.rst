Classificar imagens
===================

Classifique imagens com os classificadores internos ou personalizados.

Parãmetros
~~~~~~~~~~

O método Classificar imagens aceita os seguintes parãmetros para a sua execução:

versionDate

    A data da versão da API REST.

apiKey

    A chave da API.

endPoint

    O novo endPoint da API.

headers

    Nome do cabeçalho que será usado na requisição HTTP.

options

    Opções do Classificar imagens. Saiba mais: https://www.ibm.com/watson/developercloud/visual-recognition/api/v3/#classify_an_image
