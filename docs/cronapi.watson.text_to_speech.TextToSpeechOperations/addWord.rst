Adicionar uma palavra personalizada
===================================

O objeto Java que terá uma palavra adicionada.

Parãmetros
~~~~~~~~~~

O método Adicionar uma palavra personalizada aceita os seguintes parãmetros para a sua execução:

username

    Nome do usuário.

password

    A senha.

endPoint

    O novo endPoint da API.

headers

    Nome do cabeçalho que será usado na requisição HTTP.

model

    O objeto Java ao qual uma palavra deve ser adicionada.

translation

    Um objeto Java que fornece informações sobre a palavra a ser adicionada ao modelo de voz personalizado.
