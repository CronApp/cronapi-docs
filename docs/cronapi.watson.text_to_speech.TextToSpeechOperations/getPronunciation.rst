Obter pronúncia
===============

Retorna a pronúncia fonética para a palavra especificada.

Parãmetros
~~~~~~~~~~

O método Obter pronúncia aceita os seguintes parãmetros para a sua execução:

username

    Nome do usuário.

password

    A senha.

endPoint

    O novo endPoint da API.

headers

    Nome do cabeçalho que será usado na requisição HTTP.

word

    A palavra para a qual a pronúncia é solicitada.

voice

    Um objeto Java que especifica o idioma em que a pronúncia deve ser retornada.

phoneme

    O objeto Java para o formato de fonema no qual a pronúncia deve ser retornada.

customizationId

    O GUID de um modelo de voz personalizado para o qual a pronuncia deve ser retornada.
