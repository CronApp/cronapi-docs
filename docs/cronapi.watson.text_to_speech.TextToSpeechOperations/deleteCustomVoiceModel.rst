Excluir um modelo personalizado
===============================

Exclui o modelo de voz personalizado especificado.

Parãmetros
~~~~~~~~~~

O método Excluir um modelo personalizado aceita os seguintes parãmetros para a sua execução:

username

    Nome do usuário.

password

    A senha.

endPoint

    O novo endPoint da API.

headers

    Nome do cabeçalho que será usado na requisição HTTP.

model

    O objeto Java que deve ser excluído.
