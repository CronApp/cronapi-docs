Receber vozes
=============

Recupera uma lista de todas as vozes disponíveis para uso com o serviço.

Parãmetros
~~~~~~~~~~

O método Receber vozes aceita os seguintes parãmetros para a sua execução:

username

    Nome do usuário.

password

    A senha.

endPoint

    O novo endPoint da API.

headers

    Nome do cabeçalho que será usado na requisição HTTP.
