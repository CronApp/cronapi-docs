Excluir uma palavra personalizada
=================================

Exclui uma única palavra do modelo de voz personalizado especificado.

Parãmetros
~~~~~~~~~~

O método Excluir uma palavra personalizada aceita os seguintes parãmetros para a sua execução:

username

    Nome do usuário.

password

    A senha.

endPoint

    O novo endPoint da API.

headers

    Nome do cabeçalho que será usado na requisição HTTP.

model

    O objeto Java a partir do qual uma palavra deve ser excluída.

translation

    Um objeto Java que especifica a palavra que deve ser excluída do modelo de voz personalizado.
