Analisar
========

Analise as características do idioma natural.

Parãmetros
~~~~~~~~~~

O método Analisar aceita os seguintes parãmetros para a sua execução:

versionDate

    A data da versão da API REST.

username

    Nome do usuário

password

    A senha

endPoint

    O novo endPoint da API.

headers

    Nome do cabeçalho que será usado na requisição HTTP.

options

    Opções do Analisar. Saiba mais:  https://www.ibm.com/watson/developercloud/natural-language-understanding/api/v1/#methods
