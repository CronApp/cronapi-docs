Analisar o tom geral
====================

Use o EndPoint geral para analisar o tom do seu conteúdo de entrada.

Parãmetros
~~~~~~~~~~

O método Analisar o tom geral aceita os seguintes parãmetros para a sua execução:

versionDate

    A data da versão da API REST para usar.

username

    Nome do usuário.

password

    A senha.

endPoint

    O novo endPoint da API.

headers

    Nome do cabeçalho que será usado na requisição HTTP.

options

    Um objeto Java que especifica os parâmetros da solicitação. Saiba mais: https://www.ibm.com/watson/developercloud/tone-analyzer/api/v3/#post-tone
