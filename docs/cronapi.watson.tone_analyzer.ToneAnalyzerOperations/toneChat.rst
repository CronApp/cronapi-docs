Analisar o tom emocional do cliente
===================================

Use o EndPoint do cliente para analisar o tom e as conversas de atendimento ao cliente. Para cada enunciado de uma conversa, o método relata o subconjunto mais prevalente dos seguintes sete tons: triste, frustrado, satisfeito, animado, educado, não cortês e simpático.

Parãmetros
~~~~~~~~~~

O método Analisar o tom emocional do cliente aceita os seguintes parãmetros para a sua execução:

this

    null

versionDate

    A data da versão da API REST para usar.

username

    Nome do usuário.

password

    A senha.

endPoint

    O novo endPoint da API.

headers

    Nome do cabeçalho que será usado na requisição HTTP.

options

    Um objeto Java que contém o conteúdo a ser analisado. Saiba mais: https://www.ibm.com/watson/developercloud/tone-analyzer/api/v3/#customer-tone
