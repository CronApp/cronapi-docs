Obter coluna de dados
=====================

Função para obter uma lista de dados da coluna. Exemplo: lista de todos os e-mails de usuários

Parãmetros
~~~~~~~~~~

O método Obter coluna de dados aceita os seguintes parãmetros para a sua execução:

ds

    Fonte de dados

Nome do campo

    Nome do campo
