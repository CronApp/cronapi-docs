Adicionar palavras personalizadas
=================================

Adiciona uma ou mais palavras e suas traduções ao modelo de voz personalizado especificado.

Parãmetros
~~~~~~~~~~

O método Adicionar palavras personalizadas aceita os seguintes parãmetros para a sua execução:

username

    Nome do usuário.

password

    A senha.

endPoint

    O novo endPoint da API.

headers

    Nome do cabeçalho que será usado na requisição HTTP.

model

    O objeto Java ao qual as palavras devem ser adicionadas.

translations

    Uma lista separada por vírgulas de um ou mais objetos Java que fornece informações sobre as palavras a serem adicionadas ao modelo de voz personalizado.
