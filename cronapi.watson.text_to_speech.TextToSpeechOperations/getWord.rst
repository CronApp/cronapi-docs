Listar uma palavra personalizada
================================

Retorna a tradução para uma única palavra a partir do modelo personalizado especificado.

Parãmetros
~~~~~~~~~~

O método Listar uma palavra personalizada aceita os seguintes parãmetros para a sua execução:

username

    Nome do usuário.

password

    A senha.

endPoint

    O novo endPoint da API.

headers

    Nome do cabeçalho que será usado na requisição HTTP.

model

    O objeto Java que contém a palavra que deve ser retornada.

word

    A palavra que deve ser devolvida do modelo de voz personalizado.
