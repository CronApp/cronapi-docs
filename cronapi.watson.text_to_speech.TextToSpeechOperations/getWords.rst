Listar palavras personalizadas
==============================

Lista todas as palavras e suas traduções para o modelo de voz personalizado especificado.

Parãmetros
~~~~~~~~~~

O método Listar palavras personalizadas aceita os seguintes parãmetros para a sua execução:

username

    Nome do usuário.

password

    A senha.

endPoint

    O novo endPoint da API.

headers

    Nome do cabeçalho que será usado na requisição HTTP.

model

    O objeto Java cujas palavras devem ser listadas.
