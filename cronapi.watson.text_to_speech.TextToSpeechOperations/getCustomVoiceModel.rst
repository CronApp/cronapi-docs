Listar um modelo personalizado
==============================

Lista todas as informações sobre um modelo de voz personalizado especificado.

Parãmetros
~~~~~~~~~~

O método Listar um modelo personalizado aceita os seguintes parãmetros para a sua execução:

username

    Nome do usuário.

password

    A senha.

endPoint

    O novo endPoint da API.

headers

    Os pares de cabeçalhos do nome da API

model

    O objeto Java que deve ser retornado.
