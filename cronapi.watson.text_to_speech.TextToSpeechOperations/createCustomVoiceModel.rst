Criar um modelo personalizado
=============================

Cria um novo modelo de voz personalizado vazio.

Parãmetros
~~~~~~~~~~

O método Criar um modelo personalizado aceita os seguintes parãmetros para a sua execução:

username

    Nome do usuário.

password

    A senha.

endPoint

    O novo endPoint da API.

headers

    Nome do cabeçalho que será usado na requisição HTTP.

name

    Uma descrição do novo modelo de voz personalizado.

language

    O idioma do novo modelo de voz personalizado.

description

    Uma descrição do novo modelo de voz personalizado.
