Excluir modelo
==============

Exclua um modelo de tradução personalizado.

Parãmetros
~~~~~~~~~~

O método Excluir modelo aceita os seguintes parãmetros para a sua execução:

username

    O nome de usuário.

password

    A senha.

endPoint

    O novo endPoint da API.

headers

    Os cabeçalhos padrão para serem usados em todas as solicitações HTTP.

deleteModelOptions

    Exclua as opções do modelo.  Saiba mais: https://www.ibm.com/watson/developercloud/language-translator/api/v2/#delete-model
