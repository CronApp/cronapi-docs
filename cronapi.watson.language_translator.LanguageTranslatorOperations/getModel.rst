Obter detalhes do modelo
========================

Obtenha informações para um modelo de tradução, incluindo o status de treinamento para modelos personalizados.

Parãmetros
~~~~~~~~~~

O método Obter detalhes do modelo aceita os seguintes parãmetros para a sua execução:

username

    O nome de usuário.

password

    A senha.

endPoint

    O novo endPoint da API.

headers

    Os cabeçalhos padrão para serem usados em todas as solicitações HTTP.

getModelOptions

    Exclua as opções do modelo. Saiba mais: https://www.ibm.com/watson/developercloud/language-translator/api/v2/#get-model
