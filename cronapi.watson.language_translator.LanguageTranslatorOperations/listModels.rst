Listar de modelos
=================

Lista os modelos de tradução disponíveis.

Parãmetros
~~~~~~~~~~

O método Listar de modelos aceita os seguintes parãmetros para a sua execução:

username

    O nome de usuário.

password

    A senha.

endPoint

    O novo endPoint da API.

headers

    Os cabeçalhos padrão para serem usados em todas as solicitações HTTP.
