Obter perfil
============

Gera um perfil de personalidade para o autor do texto de entrada.

Parãmetros
~~~~~~~~~~

O método Obter perfil aceita os seguintes parãmetros para a sua execução:

versionDate

    A data da versão da API REST.

username

    O nome de usuário.

password

    A senha.

endPoint

    O novo endPoint da API.

headers

    Os cabeçalhos padrão para serem usados em todas as solicitações HTTP.

profileOptions

    As opções para a chamada. Saiba mais: https://www.ibm.com/watson/developercloud/personality-insights/api/v3/#profile
