Enviar notificação
==================

Função que envia notificação para um dispositivo movel usando o FCM (Firebase Cloud Message).

Parãmetros
~~~~~~~~~~

O método Enviar notificação aceita os seguintes parãmetros para a sua execução:

serverKey

    Chave do Servidor

paramTo

    Destinatário

paramTitle

    Título

paramBody

    Conteúdo

paramData

    {{FirebaseData}}
