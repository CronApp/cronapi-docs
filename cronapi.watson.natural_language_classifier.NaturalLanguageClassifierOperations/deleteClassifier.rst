Eliminar classificador
======================

Eliminar classificador.

Parãmetros
~~~~~~~~~~

O método Eliminar classificador aceita os seguintes parãmetros para a sua execução:

username

    O nome de usuário.

password

    A senha.

endPoint

    O novo endPoint da API.

headers

    Os cabeçalhos padrão para serem usados em todas as solicitações HTTP.

classifierId

    ID do classificador para deletar.
