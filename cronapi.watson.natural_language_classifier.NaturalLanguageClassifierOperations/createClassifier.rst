Criar classificador
===================

Envia dados para criar e treinar um classificador e retorna informações sobre o novo classificador.

Parãmetros
~~~~~~~~~~

O método Criar classificador aceita os seguintes parãmetros para a sua execução:

username

    O nome de usuário.

password

    A senha.

endPoint

    O novo endPoint da API.

headers

    Os cabeçalhos padrão para serem usados em todas as solicitações HTTP.

name

    Nome fornecido pelo usuário para o classificador.

language

    O idioma utilizado para o classificador.

trainingData

    Metadados no formato JSON. Os metadados identificam o idioma dos dados e um nome opcional para identificar o classificador.
