Listar uma palavra personalizada
================================

Lista informações sobre uma palavra personalizada de um modelo de idioma personalizado.

Parãmetros
~~~~~~~~~~

O método Listar uma palavra personalizada aceita os seguintes parãmetros para a sua execução:

username

    Nome do usuário.

password

    A senha.

endPoint

    O novo endPoint da API.

headers

    Nome do cabeçalho que será usado na requisição HTTP.

customizationId

    O GUID do modelo de idioma personalizado a partir do qual uma palavra deve ser consultada.

wordName

    A palavra personalizada que deve ser consultada a partir do modelo personalizado.
