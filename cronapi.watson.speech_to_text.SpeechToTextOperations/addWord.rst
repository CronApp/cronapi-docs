Adicionar uma palavra personalizada
===================================

Adiciona uma palavra personalizada a um modelo de idioma personalizado.

Parãmetros
~~~~~~~~~~

O método Adicionar uma palavra personalizada aceita os seguintes parãmetros para a sua execução:

username

    Nome do usuário.

password

    A senha.

endPoint

    O novo endPoint da API.

headers

    Nome do cabeçalho que será usado na requisição HTTP.

customizationId

    O GUID do modelo de idioma personalizado ao qual uma palavra deve ser adicionada.

word

    Um objeto Java que fornece informações sobre a palavra personalizada a ser adicionada.
