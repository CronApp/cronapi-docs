Listar um modelo de idioma personalizado
========================================

Lista informações sobre o modelo personalizado de idioma  especificado.

Parãmetros
~~~~~~~~~~

O método Listar um modelo de idioma personalizado aceita os seguintes parãmetros para a sua execução:

username

    Nome do usuário.

password

    A senha.

endPoint

    O novo endPoint da API.

headers

    Nome do cabeçalho que será usado na requisição HTTP.

customizationId

    O GUID do modelo de idioma personalizado para qual informação deve ser devolvida.
