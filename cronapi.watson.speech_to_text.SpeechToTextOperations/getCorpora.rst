Listar corpos
=============

Lista informações sobre todos os corpos a partir de um modelo de idioma personalizado.

Parãmetros
~~~~~~~~~~

O método Listar corpos aceita os seguintes parãmetros para a sua execução:

username

    Nome do usuário.

password

    A senha.

endPoint

    O novo endPoint da API.

headers

    Nome do cabeçalho que será usado na requisição HTTP.

customizationId

    O GUID do modelo de idioma personalizado para o qual os corpos devem ser listados.
