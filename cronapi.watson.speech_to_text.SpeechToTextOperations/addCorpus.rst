Adicionar um corpus
===================

Adiciona um único arquivo de texto corpus com novos dados de treinamento para um modelo de idioma personalizado.

Parãmetros
~~~~~~~~~~

O método Adicionar um corpus aceita os seguintes parãmetros para a sua execução:

username

    Nome do usuário.

password

    A senha.

endPoint

    O novo endPoint da API.

headers

    Nome do cabeçalho que será usado na requisição HTTP.

customizationId

    O GUID do modelo de linguagem personalizado ao qual um corpus deve ser adicionado.

corpusName

    O nome do corpus que deve ser adicionado.

corpusFile

    Um arquivo de texto simples que contém os dados de treinamento para o corpus.

allowOverwrite

    Indica se o corpus especificado é para substituir um corpus existente com o mesmo nome.
