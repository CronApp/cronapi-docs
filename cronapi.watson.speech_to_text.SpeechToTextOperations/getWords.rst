Listar palavras personalizadas
==============================

Lista informações sobre palavras personalizadas de um modelo de idioma personalizado.

Parãmetros
~~~~~~~~~~

O método Listar palavras personalizadas aceita os seguintes parãmetros para a sua execução:

username

    Nome do usuário.

password

    A senha.

endPoint

    O novo endPoint da API.

headers

    Nome do cabeçalho que será usado na requisição HTTP.

customizationId

    O GUID do modelo de idioma personalizado a partir do qual as palavras devem ser consultadas.

type

    O tipo de palavras a serem listadas a partir do recurso de palavras do modelo de idioma personalizado.

sort

    A ordem em que as palavras devem ser listadas.
