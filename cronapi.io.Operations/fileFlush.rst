Limpar arquivo
==============

Função para limpar arquivo. Esta função recebe um arquivo aberto para a escrita e limpa, ou apaga, todo o conteúdo, tornado o arquivo vazio.

Parâmetros
~~~~~~~~~~

O método Limpar arquivo aceita os seguintes parãmetros para a sua execução:

Arquivo aberto para escrita

     Retorno da função "Abrir arquivo para escrita"
