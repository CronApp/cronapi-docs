Copiar arquivo
==============

Função para copiar o arquivo do caminho de destino para o caminho de origem

Parâmetros
~~~~~~~~~~

O método Copiar arquivo aceita os seguintes parãmetros para a sua execução:

Caminho de origem

     Caminho completo de origem do arquivo. Exemplo: /MinhaPasta/meuArquivo.txt

pathTo

    Caminho de destino
