Criar pasta
===========

Função para criar nova pasta. Exemplo: /CAMINHO/NOME_DA_PASTA

Parâmetros
~~~~~~~~~~

O método Criar pasta aceita os seguintes parãmetros para a sua execução:

Caminho

     Caminho que deve ser criado a pasta. Exemplo: PASTA_TEMPORARIA/

Nome da pasta

     Exemplo: MinhaPasta
