Download de arquivo a partir de URL
===================================

Função para fazer download de arquivo a partir de URL e salvar o conteúdo em um arquivo

Parâmetros
~~~~~~~~~~

O método Download de arquivo a partir de URL aceita os seguintes parãmetros para a sua execução:

Endereço URL

     Endereço completo do arquivo. Exemplo: https://www.cronapp.io

Pasta para salvar

     Caminho completo da pasta para salvar o arquivo. Exemplo: /MinhaPasta/

Nome do arquivo

     Nome do arquivo sem extensão. Exemplo: MeuArquivo

Extensão do arquivo

     Extensão do arquivo com ponto. Exemplo: .html or .txt or .extensao
