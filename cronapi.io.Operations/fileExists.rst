Existe arquivo?
===============

Função para verificar se existe o arquivo. Retorna verdadeiro caso exsita, se não existir retorna falso

Parâmetros
~~~~~~~~~~

O método Existe arquivo? aceita os seguintes parãmetros para a sua execução:

Caminho do arquivo

     Exemplo: PASTA_TEMPORARIA/MeuArquivo.txt
